# دليل البدء السريع ⚡

## للتشغيل السريع:

### 1. افتح Terminal في مجلد المشروع

### 2. ثبت الحزم:
```bash
npm run install-all
```

### 3. شغّل التطبيق:
```bash
npm run dev
```

### 4. افتح المتصفح:
اذهب إلى: **http://localhost:5173**

---

## إذا لم يعمل:

### تحقق من:
1. ✅ Node.js مثبت (الإصدار 16+)
2. ✅ جميع الحزم مثبتة
3. ✅ المنفذ 3000 و 5173 غير مستخدمين

### حل المشاكل:

**خطأ: "Cannot find module"**
```bash
cd server && npm install
cd ../client && npm install
```

**خطأ: "Port already in use"**
- أغلق البرامج الأخرى
- أو غيّر المنفذ في `server/index.js` و `client/vite.config.js`

**السيرفر لا يعمل:**
```bash
cd server
npm run dev
```

**الواجهة لا تعمل:**
```bash
cd client
npm run dev
```
